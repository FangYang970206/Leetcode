# 问题描述

二叉搜索树中的两个节点被错误地交换。

请在不改变其结构的情况下，恢复这棵树。

# 例子

```bash
输入: [1,3,null,null,2]

   1
  /
 3
  \
   2

输出: [3,1,null,null,2]

   3
  /
 1
  \
   2


输入: [3,1,4,null,null,2]

  3
 / \
1   4
   /
  2

输出: [2,1,4,null,null,3]

  2
 / \
1   4
   /
  3
```

# 方法

既然是二叉搜索树，那么它便符合中序遍历的有序性，任意调换两个数，会有这样的特征，出现两处有序性不符的地方，这时可以定位调换的元素了，直接调换即可。

注意，这里的python解决方案有问题，由于python的复制引用特性，烦。
