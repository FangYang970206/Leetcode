# 问题描述

编写一个 SQL 查询，查找所有至少连续出现三次的数字。

```bash
+----+-----+
| Id | Num |
+----+-----+
| 1  |  1  |
| 2  |  1  |
| 3  |  1  |
| 4  |  2  |
| 5  |  1  |
| 6  |  2  |
| 7  |  2  |
+----+-----+
```

例如，给定上面的 Logs 表， 1 是唯一连续出现至少三次的数字。

```bash
+-----------------+
| ConsecutiveNums |
+-----------------+
| 1               |
+-----------------+
```

# 方法

这题需要用三个别名表示logs，比较三个连续数字是否相同，DISTINCT关键字则是用来去重的，因为可能会出现相同连续数字不止一次。
