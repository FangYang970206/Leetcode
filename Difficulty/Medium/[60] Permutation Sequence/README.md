# 问题描述

给出集合 [1,2,3,…,n]，其所有元素共有 n! 种排列。

按大小顺序列出所有排列情况，并一一标记，当 n = 3 时, 所有排列如下：

"123"
"132"
"213"
"231"
"312"
"321"
给定 n 和 k，返回第 k 个排列。

说明：

给定 n 的范围是 [1, 9]。
给定 k 的范围是[1,  n!]。

# 例子

```bash
输入: n = 3, k = 3
输出: "213"

输入: n = 4, k = 9
输出: "2314"
```

# 方法

方法是根据k/(n-1)!来确定下一个数的位置，这里有一点要小心，第一次k/(n-1)!需要将k减去1，因为从1开始计数，这是个大坑。另外，每次添加字符后，需要把添加的字符在原来数组中删除。
